# Sub-6 Predicts mmWave Beam-forming Vectors:
Using sub-6 GHz channels to predict top-n beam-forming vectors for 28 GHz users.

# Requirements:

Essential:

1- MATLAB deep learning toolbox.

2- [DeepMIMO dataset](http://www.deepmimo.net/?i=1)

Optional:

1- NVIDIA GPU card.

2- CUDA toolkit.

3- cuDNN package.
